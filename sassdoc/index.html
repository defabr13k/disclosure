    dc-slide-ratios

    $dc-slide-ratios: (
  'minor_second'  : 15/16,
	'major_second'  : 8/9,
	'minor_third'   : 5/6,
	'major_third'   : 4/5,
  'perfect_fourth': 3/4,
  'perfect_fifth' : 2/3,
  'minor_sixth'   : 5/8,
  'golden_section': 1/1.618,
  'major_sixth'   : 3/5,
  'minor_seventh' : 9/16,
  'major_seventh' : 8/15,
  'octave'        : 1/2,
  'major_tenth'   : 2/5,
  'major_eleventh': 3/8,
  'major_twelfth' : 1/3,
  'octave_double' : 1/4
) !default;

    Description

    Slide ratios equal to modular scale ratios

    Type

    Map

    ms-ratio

    $ms-ratio: map-get($ms-ratios, 'minor_seventh') !default;

    Description

    Slide Ratio based on http://modularscale.com/scale/?px1=16&px2=18&ra1=1.333&ra2=0

    Type

    Number
